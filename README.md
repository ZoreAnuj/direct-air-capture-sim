# Direct Air Capture Simulation

This project simulates direct air capture (DAC) and carbon removal processes. It exists to model and explore the technical parameters and performance of carbon capture technology in a controlled, synthetic environment.

## Key Features
*   Simulates core DAC adsorption and desorption cycles.
*   Models key performance metrics like energy consumption and capture rates.
*   Provides a configurable instrument for testing different operational parameters.
*   Includes basic data visualization for simulation outputs.

## Tech Stack
*   Python
*   NumPy / Pandas
*   Matplotlib

## Getting Started
1.  Clone the repository: `git clone https://github.com/zoreanuj/direct-air-capture-sim.git`
2.  Install dependencies: `pip install -r requirements.txt`
3.  Run the main simulation: `python src/simulator.py`
